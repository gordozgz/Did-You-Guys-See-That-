DdSTT — трей-приложение с рандомным показом GIF на весь экран
================================================================

Что делает:
- Работает в фоне, с собственной иконкой в трее (тёмный кружок
  с "лицом-призраком", видна всегда).
- Через случайный промежуток времени: минимум статичный — 30 секунд
  (MIN_INTERVAL_SEC), максимум — 30 минут (MAX_INTERVAL_SEC); каждый
  цикл пауза выбирается случайно между ними. Меняются в начале файла
  ddstt_tray.py. Один раз проигрывает
  assets\DdSTT.gif на весь экран с сохранением альфа-прозрачности
  (используется настоящее Windows layered window, а не просто
  "прозрачный цвет" — края гифки останутся сглаженными).
- Сразу после гифки на 3 секунды показывается чёрно-белый эффект
  (снимок рабочего стола в оттенках серого во весь экран). Это чисто
  визуальный оверлей — клики и движения мыши проходят сквозь него,
  ничего не блокируется. Длительность/вкл-выкл: BW_EFFECT_ENABLED,
  BW_EFFECT_DURATION_SEC.
- После этого таймер сбрасывается и ждёт заново — бесконечно.
- Обратный отсчёт до следующего показа виден:
    * во всплывающей подсказке — наведите мышь на иконку в трее;
    * в самом меню трея первым пунктом.
- В трее:
    * "След. показ через: MM:SS" — текущий таймер (не кликабельно)
    * Force run (проверка)  — играет гифку один раз сразу; повторное
      нажатие снова проигрывает с нуля (пока идёт показ, повторные
      клики игнорируются, а не встают в очередь)
    * Включено               — вкл/выкл случайные показы (пауза)
    * Автозапуск              — вкл/выкл запуск вместе с Windows
    * Выход

Автозапуск:
- При первом запуске приложение само добавляет себя в реестр
  (HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Run),
  без прав администратора, без ярлыков и установщиков.
- Отключить/включить можно в любой момент через меню трея.

Как запустить (без сборки в exe):
    1. Установить Python 3.9+ для Windows.
    2. pip install -r requirements.txt
    3. Запустить: pythonw ddstt_tray.py
       (pythonw — чтобы не было чёрного окна консоли;
        для отладки используйте обычный python ddstt_tray.py)

Как собрать в единый .exe (рекомендуется для "шуточной проги"):
    1. Установить Python 3.9+ для Windows.
    2. Запустить build_exe.bat (он сам поставит pyinstaller и зависимости).
    3. Готовый файл появится в dist\DdSTT.exe — можно кидать другу,
       Python у него не нужен.
    4. При первом запуске exe сам пропишется в автозагрузку.

Важно:
- Всё собирается и тестируется ТОЛЬКО на Windows (используется
  нативный WinAPI UpdateLayeredWindow через pywin32). На Linux/macOS
  не запустится.
- Если захотите заменить гифку — просто замените файл
  assets\DdSTT.gif на свой (с альфа-каналом) и пересоберите exe.
- Растяжение на весь экран включено по умолчанию
  (STRETCH_FULLSCREEN = True). Если нужно показывать гифку в
  реальном размере по центру экрана — поставьте False.

Структура проекта:
    ddstt_tray.py     — основной код
    assets\DdSTT.gif  — гифка
    requirements.txt  — зависимости для pip
    build_exe.bat     — сборка в один exe-файл
    README.txt        — этот файл
